## 🚀 快速开始

### 1. 查看图标
```bash
cd packages/icons/examples/vue-demo
npm install
npm run dev
# 访问 http://localhost:5004
```

### 2. 使用图标

**Vue**:
```vue
<HomeIcon size="24" :strokeWidth="2" />
```

**React**:
```tsx
<HomeIcon size={24} strokeWidth={2} />
```

**Lit**:
```html
<ld-icon-home size="24" stroke-width="2"></ld-icon-home>
```
